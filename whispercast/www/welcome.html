Welcome to whisperCast

Please use one of the items on the left to go through some examples.

The root of these examples is at:
$INSTALL_DIR/whispercast/media/www

The flags for running the demo applications are to be found under:
$INSTALL_DIR/whispercast/etc/ini/demo/
The runtime configuration files for the demo applications are to be found under:
$INSTALL_DIR/whispercast/etc/config/demo/

There are several configuration files:
  • The demo configuration file can be found at:
    $INSTALL_DIR/whispercast/etc/config/demo/whispercast.config
    $INSTALL_DIR/whispercast/etc/config/demo/whisperer.config
    Use the admin interface to modify these elements (user: admin, password: test).
  • The flags for running the demo can be found at:
    $INSTALL_DIR/whispercast/etc/ini/demo/whispercast.ini
    $INSTALL_DIR/whispercast/etc/ini/demo/whisperer.ini
    Please feel free to experiment with the various options here (run: $INSTALL_DIR/whispercast/bin/whispercast --help or $INSTALL_DIR/whispercast/bin/whisperer --help for a full list of all command line flags).
Random notes:
  • The data in the runtime configuration files is JSON encoded, so be careful with all the braces, slashes and escaping when modifying them.
  • In order to change the admin password you should modify the --admin_passwd flag.
  • To disable the auto-generated admin interface, remove the --rpc_js_form_path flag.
  • By default, the ports on which whispercast is listening are:
    8880 - HTTP
    8935 - RTMP
    The port on which whisperer is listening is:
    8881 - HTTP

In order to view the current element configuration, you should go to admin interface, choose Global Config and inside there are three important parts to figure out the config:

  • GetElementConfig - to figure out the elements in the configuration (use Send Command button to get a JSON encoded version of the curent config)
  • GetElementExports - to find out what media names are exported (http & rtmp)
  • GetSavesConfig - to check what savers are configured (none should be in the standard demo cofiguration)
